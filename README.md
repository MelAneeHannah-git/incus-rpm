# Incus RPM Build Project for AlmaLinux 10

This repository contains the necessary files and instructions to build RPMs for:
- raft (v0.17.1)
- cowsql (v1.15.6)
- incus (v6.8.0)

## Requirements
- AlmaLinux 10
- RPM build tools
- Development tools

## Directory Structure
- SPECS/ - RPM spec files
- SOURCES/ - Source tarballs
- RPMS/ - Built RPMs
- docs/ - Documentation and build guides
